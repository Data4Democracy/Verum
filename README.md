## How do I use this?

### To run the crawler

From the project directory, navigate to the `ncleg/spiders` directory, and use the scrapy commandline tool to launch the spider.

For example:

```bash
cd ncleg/spiders
scrapy crawl test
# or 
scrapy runspider vhr_spider.py
```
